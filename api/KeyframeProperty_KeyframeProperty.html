Class: KeyframeProperty

KeyframeProperty(millisecond, name, value, easingopt)

KeyframeProperty

Constructor

new KeyframeProperty(millisecond, name, value, easingopt)

Represents an individual component of an actor's keyframe state. In most cases you won't need to deal with this object directly, as the {{#crossLink "Actor"}}{{/crossLink}} APIs abstract a lot of what this Object does away for you.

Parameters:
Name Type Attributes Description
millisecond number

Where on the animation timeline this KeyframeProperty is.
name string

The property's name, such as "x" or "opacity".
value number | string | function

The value that this KeyframeProperty represents.
easing string <optional>

The easing curve at which this KeyframeProperty should be animated to. Defaults to "linear".
